# 🚀 Sobre Mim / My Journey

Olá! Bem-vindo(a) ao meu repositório. 

Atualmente, estou imerso em um treinamento **Full Stack**, onde busco construir uma base sólida tanto em **Front-end** quanto em **Back-end**, além de me familiarizar com as principais ferramentas do ecossistema de desenvolvimento de software.

### 🎯 Minha Filosofia de Aprendizado: "T-Shaped Skills"
Acredito fortemente no conceito de ter uma visão generalista para entender o todo, mas com uma especialização profunda. Por isso, embora esteja absorvendo conhecimento de ponta a ponta (Full Stack) para compreender a dor e o fluxo dos desenvolvedores, meu foco principal e objetivo de carreira é a área de **DevOps**.

> *"Entender a base de tudo para entregar excelência no que decidi focar."*

### 🛠️ No que estou focado:
* **Entendimento de Software:** Construção de APIs, lógica de negócios (Back-end) e interfaces (Front-end).
* **Cultura DevOps:** Automação, CI/CD, infraestrutura como código e deploy ágil.
